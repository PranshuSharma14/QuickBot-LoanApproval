## üöÄ Quick Start

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. **Install Dependencies**
   ```bash
   cd frontend
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Open Your Browser**
   ```
   http://localhost:3000
   ```

The frontend will automatically connect to the backend at `http://127.0.0.1:8000`
